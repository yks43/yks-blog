## 设计模式

### 单例模式
> 一个类只能有一个实例

编写步骤：
* 将构造函数私有化
* 在类的内部创建实例
* 提供获取唯一实例的方法

#### 懒汉式
``` java
public class Yks {

    // 1.将构造函数私有化，不可以通过new的方式来创建对象
    private Yks(){}

    // 2.在类的内部创建自行实例
    private static Yks yks  = new Yks();

    // 3.提供获取唯一实例的方法
    public static Yks getJava3y() {
        return yks;
    }
}
```
